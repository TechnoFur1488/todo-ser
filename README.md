# todo-serv

## 📋 Requirements

- [Docker](https://www.docker.com/) (version 20.10.0+)
- [Docker Compose](https://docs.docker.com/compose/)
- [Node.js](https://nodejs.org/) (version 18+)
- [Git](https://git-scm.com/)

## 🚀 Quick Start

```bash copy
    git clone https://github.com/TechnoFur1488/todo-ser.git

    cd todo-ser

    npm i

    npm run dev
```
